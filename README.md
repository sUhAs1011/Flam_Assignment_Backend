# Flam_Assignment_Backend

A lightweight, production-style background job processing system with:

- ✅ Persistent SQLite storage
- ✅ Multi-worker execution (Unix fork)
- ✅ Retry with exponential backoff
- ✅ Dead Letter Queue (DLQ)
- ✅ CLI interface for job + worker management
- ✅ Config system (max retries, backoff, timeout)
- ✅ Graceful shutdown
- ✅ Per-job logs
